## 1. Objective

Build the first public business profile page.

Users should be able to visit:

/business/:slug

and view the corresponding business profile fetched from Supabase.

---

## 2. Existing Context

The project already includes:

- Business Profile Form
- Profile Preview
- Save Profile
- Supabase integration
- business_profiles table
- slug generation
- getBusinessProfileBySlug(slug)

Reuse the existing retrieval function.

---

## 3. Build

Create a new public route:

/business/:slug

When opened:

- Read the slug from the URL.
- Fetch the profile using getBusinessProfileBySlug().
- Display the business profile using the existing profile layout.

Do not duplicate UI.

Reuse existing components wherever possible.

---

## 4. Loading State

Display a centered loading indicator.

## 5. Error State
