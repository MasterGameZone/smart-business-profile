## 3. Build

Create the initial Supabase database schema.

Create one table:

business_profiles

Columns:

id
- UUID
- Primary Key
- Default: gen_random_uuid()

business_name
- TEXT
- Required

owner_name
- TEXT
- Required

business_category
- TEXT
- Required

phone_number
- TEXT
- Required

whatsapp_number
- TEXT
- Optional

email
- TEXT
- Optional

website
- TEXT
- Optional

address
- TEXT
- Optional

about_business
- TEXT
- Optional

logo_url
- TEXT
- Optional

created_at
- TIMESTAMP WITH TIME ZONE
- Default: now()

updated_at
- TIMESTAMP WITH TIME ZONE
- Default: now()
