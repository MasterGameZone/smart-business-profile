## 5. QR Code Improvements

Keep the existing QR Code.

Add two new actions below it.

### Download QR Code

Create a button:

Download QR Code

Behaviour:

- Download the QR Code as a PNG image.
- Use a clean filename:

business-profile-qr.png

---

### Share QR Code

Create another button:

Share QR Code

Behaviour:

If the browser supports sharing image files using the Web Share API:

- Share the QR Code image.

Otherwise:

- Automatically download the QR Code image.
- Display a message:

"Your browser doesn't support direct QR sharing. The QR Code has been downloaded instead."
