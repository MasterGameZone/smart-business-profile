## 1. Objective

Build Version 0.4 of Smart Business Profile.

The objective is to generate a live Business Profile Preview using the information entered in the form.

This version introduces the first functional digital business profile.

Do NOT save any data permanently.

- React + Vite + TypeScript
- Tailwind CSS
- React Router
- Landing Page
- Business Profile Form
- Form Validation

Continue working inside the existing GitHub-imported project.

## 3. Build

When the user successfully clicks **Continue**, navigate to a new route:

/profile-preview

Pass the entered form data using React state (or another temporary in-memory method).

- Local Storage

- Backend

## 4. Business Profile Preview

Create a professional responsive digital business card.

Display:

- Business Logo (if selected)
- Business Name
- Owner Name
- Business Category
- Phone Number
- WhatsApp Number
- Email Address
- Website
- Address
- About Business

If optional fields are empty, simply hide them.

- Call
- WhatsApp
- Email
