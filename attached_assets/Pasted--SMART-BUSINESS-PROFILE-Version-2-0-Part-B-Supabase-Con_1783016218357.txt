## Tasks

1. Install the official Supabase JavaScript client.

2. Create a reusable Supabase client.

Example structure:

src/
  lib/
    supabase.ts

3. Read credentials from environment variables.

Create/update:

.env.example

Include only placeholders:

VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key

Do NOT hardcode any credentials.

4. Configure the Supabase client using:

import { createClient } from "@supabase/supabase-js"

Use:

import.meta.env.VITE_SUPABASE_URL
import.meta.env.VITE_SUPABASE_ANON_KEY

5. Do NOT create database tables.

6. Do NOT implement authentication.

7. Do NOT implement CRUD operations.

8. Do NOT modify existing pages.

9. Add a simple connection test that confirms the client initializes successfully without affecting the UI.

10. If any dependency is installed, explain why.
