This is the first product version.

Keep the implementation as small and reliable as possible.

- It was imported from GitHub.
- React + Vite + TypeScript is already configured.
- Tailwind CSS is already available in the project. If it is not already configured, configure it using the official Tailwind setup for Vite before styling.

- Do NOT restructure the application.
