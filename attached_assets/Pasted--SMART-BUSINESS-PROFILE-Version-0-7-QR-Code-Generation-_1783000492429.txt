## 1. Objective

Build Version 0.7 of Smart Business Profile.

The objective is to allow users to generate and share a QR Code for their business profile preview.

This version introduces QR Code functionality only.

Do NOT implement backend or permanent public profile URLs.

---

## 2. Existing Project Context

Current project includes:

- React + Vite + TypeScript
- Tailwind CSS
- React Router
- Business Profile Form
- Business Profile Preview
- React Context
- Local Storage

Continue working inside the existing project.

Do not restructure the application.

---

## 3. Build

Generate a QR Code representing the current Business Profile Preview URL.

Since there is no backend or public profile yet, generate the QR Code using the application's current preview route.

Display the QR Code inside the Profile Preview page.

If the preview data is unavailable, do not display the QR Code.

---

## 4. QR Code Section

Create a dedicated section below the profile card.

Include:

Heading:

QR Code

Description:

Scan this QR Code to open this business profile.

Display:

- QR Code
- Small border around the QR Code
- White background
- Responsive sizing

---

## 5. Share Button

Add a new button:

Share Profile

Behaviour:

If the browser supports the Web Share API:

- Share the current page URL.

Otherwise:

- Copy the current page URL to the clipboard.
- Display a success message:

Profile link copied to clipboard.
