# SMART BUSINESS PROFILE
## Version 2.5 – Business Profile Editing

## Standard Instructions

Before making any changes:

1. Read the existing codebase completely.
2. Preserve the current project architecture.
3. Reuse existing components whenever possible.
4. Modify only the files required for this version.
5. Do not refactor unrelated code.
6. Do not add extra features.
7. Do not anticipate future versions.
8. Use the existing Supabase connection.
9. Stop immediately after completing Version 2.5.

---

## 1. Objective

Build Version 2.5 of Smart Business Profile.

The objective is to allow a saved Business Profile to be loaded back into the existing form, edited, and updated in Supabase.

This is the first CRUD enhancement.

Do NOT implement authentication.

---

## 2. Existing Project Context

Current project includes:

- Business Profile Form
- Profile Preview
- Save Profile
- Public Business Profile Page
- Supabase Integration
- Slug-based profile retrieval

Continue working inside the existing project.

Do not restructure the application.

---

## 3. Build

Implement profile editing.

When viewing a Business Profile Preview, add:

Edit Profile

When clicked:

- Load the existing profile into the Business Profile Form.
- Populate every editable field.
- Preserve the existing slug.
- Preserve the existing database ID.

The form should switch into Edit Mode.

---

## 4. Update Profile

When the user clicks:

Update Profile

Perform a Supabase UPDATE instead of INSERT.

Update all editable fields.

Do NOT create a new database row.

Do NOT change:

- id
- slug
- created_at

Automatically update:

updated_at

---

## 5. Form Behaviour

Display:

Create Mode

when creating a new profile.

Display:

Edit Mode

when editing an existing profile.

Update the page heading accordingly.

## 8. Error Handling

If the update fails:

- Display a professional error toast.
- Keep the form data intact.
- Do not lose the current profile.

If the update succeeds:

Display:

"Business Profile updated successfully."
