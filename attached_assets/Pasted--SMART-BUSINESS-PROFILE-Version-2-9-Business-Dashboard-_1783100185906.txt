# SMART BUSINESS PROFILE
## Version 2.9 – Business Dashboard

## Standard Instructions

Before making any changes:

1. Read the existing codebase completely.
2. Preserve the current project architecture.
3. Reuse existing components whenever possible.
4. Modify only the files required for this version.
5. Do not refactor unrelated code.
6. Do not add extra features.
7. Do not anticipate future versions.
8. Use the existing authentication and ownership system.
9. Stop immediately after completing Version 2.9.

---

## 1. Objective

Build the first Business Dashboard.

The dashboard should become the authenticated user's central place for managing their business profile.

This version focuses on dashboard layout and profile management only.

---

## 2. Existing Project Context

The project already includes:

- Supabase Authentication
- Protected Routes
- Business Ownership
- Business Profile CRUD
- Public Business Profile Pages
- Existing Navigation

Continue working inside the existing project.

Do not restructure the application.

---

## 3. Build

Create a new protected route:

/dashboard

Only authenticated users may access it.

Unauthenticated users must be redirected to Login.

---

## 4. Dashboard Layout

Display:

Header

- Welcome message
- User email
- Logout button

Main Section

"My Business"

If the user already owns a business profile:

Display:

- Business Name
- Category
- Slug
- Created Date
- Last Updated Date

Actions:

- View Public Profile
- Edit Profile

---

## 5. Empty State

If the authenticated user does not yet own a business profile:

Display:

Title:

Create Your First Business Profile

Description:

You haven't created a business profile yet.

Button:

Create Business Profile

Navigate to the existing Create Profile page.

---

## 6. Profile Lookup

Create a reusable function:

getBusinessProfileByOwner(ownerId)

Requirements:

- Query business_profiles
- Filter using owner_id
- Return one profile or null

Reuse this function throughout the dashboard.

---

## 7. Dashboard Navigation

When authenticated:

Navigation should include:

- Dashboard
- My Business
- Logout

When anonymous:

Navigation should continue showing:

- Login
- Sign Up

Maintain the existing navigation style.
